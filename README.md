Weather-Connector-Api

A lightweight Node.js microservice that fetches real-time weather data from OpenWeatherMap and returns clean JSON responses.
This API is designed as an assignment to be used by Bino, a WhatsApp-based search assistant.

Features:

Fetches current weather, temperature, humidity, wind speed, and conditions.

Uses OpenWeatherMap API.

Routes:

Provides a single endpoint /weather that accepts a city query parameter and returns real-time weather data from OpenWeatherMap.
